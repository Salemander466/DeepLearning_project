This project was created and designed by Team 32 of Utrecht University Deep learning Course. 

To run the code first create a venv environment called venv.

The download the requirements.txt into the environment.

Then run python main.py.

This will give you 3 options: (1) , (2) , (q)

1. Runs the hyperparemeter tuning to find the best pram for the data. This code doesn't use Xtest.mat. It splits the Xtrain at the last 200 points and uses that as the validation set. The hyperprameter range can be adjusted in the def objective function inside hyperpram_tuning.py.

2. Using the best hyperparemeter generated by (1) automatically trains on the full Xtrain.mat and evaluated on Xtest.mat

3. (q) Quits the program

Evaluation

The model is evaluated using recursive 200-step forecasting. After predicting one step, the prediction is inserted back into the input window and used to predict the next step.
MAE
MSE
RMSE
R^2
